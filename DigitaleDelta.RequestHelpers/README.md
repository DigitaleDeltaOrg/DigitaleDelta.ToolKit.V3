# DigitaleDelta.RequestHelpers

A set of helpers (extensions) for dealing with OData requests and associated header parameters.

Dependencies:

- DigitaleDelta.Authentication
- DigitaleDelta.Contracts
- DigitaleDelta.ErrorHandling
- DigitaleDelta.ODataTranslator

---
## Testing

This project uses xUnit for unit and integration tests. 

---
### Running Tests

```bash
  dotnet test
```
